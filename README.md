# hash_cracker 1.0
Python Based Hash Cracker , Create hashesh and crack easily using online or offline mode.

# Setup
```
git clone https://github.com/saravana-seeker/hash_cracker.git
cd hash_cracker
pip3 install -r requirement.txt 
python3 hash_cracker.py
```
